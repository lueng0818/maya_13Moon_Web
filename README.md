# 🌌 13 Moon Synchronotron Master System

這是一個整合星際馬雅曆法 (13 Moon Calendar) 與共時化科學 (Synchronotron) 的全方位運算系統。

## ✨ 核心功能

1.  **🔮 靈魂藍圖**：計算星系印記、PSI、女神印記與五大神諭。
2.  **🏰 時間地圖**：52 天城堡、13 天波符與 20 天星際原型導航。
3.  **🌊 流年運勢**：個人流年波符與生命課題解析。
4.  **💞 關係合盤**：伴侶合盤與內在小孩印記計算 (支援雲端通訊錄)。
5.  **👑 國王預言棋盤 (Telektonon)**：
    * 每日烏龜移動路徑 (地球/戰士/天堂之旅)。
    * 13:20 與 13:28 雙羅盤校準。
    * 神諭金字塔虛擬佈陣 (GK/SP 流)。
6.  **🧠 441 共時化科學**：MCF 大師協調頻率與 BMU 基本母體單元計算。

## 🛠️ 技術架構

* **Frontend**: Streamlit
* **Language**: Python 3.9+
* **Database**: Google Sheets (via `st-gsheets-connection`)

## 🚀 如何執行

1.  安裝套件：`pip install -r requirements.txt`
2.  啟動程式：`streamlit run app.py`
